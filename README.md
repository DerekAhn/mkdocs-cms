# MkDocs-CMS

## Project Set Up

  * Node (v 0.12.7)

  ```bash
    $ npm install
  ```

  ```bash
    $ bower install
  ```

### Run development processes

  ```bash
    $ gulp
  ```

  * If gulp is running, "rs" in the terminal will restart the servers

  ```bash
    $ rs
  ```
